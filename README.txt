Repository Structure
The repository is organized into the following directories, each containing a specific area of development:

AIMing/
├── Dataset Generation/      # Scripts for creating and preprocessing datasets from the scraped ArXiv Papers(Not including datafiles for space reasons)
├── Embed Space Experiments/ # Scripts for generating, analyzing and comparing embeddings spaces(Not including datafiles for space reasons)
├── Judge Agent/             # Code for the automated evaluation agents (Not including the model files themselves for space reasons)
├── Prompt Experimentation/  # Scripts and logs for testing different prompts
└── README.txt               # You are here

For more detailed information about a specific module, please refer to the README.txt files within that directory.
